# Boombase
A starter boilerplate for developing BoomCMS themes

Features
- Bootstrap 4.1.x
- Laravel Mix
- Fontawesome 5.3.1
- ES6 Javascript Modules
- Starter templates

Instructions:
- Clone repository in themes directory
- Run ```npm install```
- Update webpack.mix.js browsersync proxy to your local url
- Run ```npm run watch```
- Happy developing!
